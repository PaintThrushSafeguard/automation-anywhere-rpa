# Automation Anywhere

## What It Does

Automation Anywhere is an enterprise robotic process automation (RPA) platform that lets organizations automate repetitive, rule-based business processes. It combines traditional RPA bots with AI-powered document processing, analytics, and process discovery.

## Core Components

**Automation 360** -- Cloud-native platform for building, deploying, and managing bots.
**Bot Agent** -- Lightweight runtime installed on target machines to execute automations.
**Control Room** -- Centralized web interface for orchestration, scheduling, and audit logging.
**IQ Bot** -- AI-driven document extraction for semi-structured and unstructured data.
